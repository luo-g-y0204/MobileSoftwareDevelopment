# Lab5：Art Space 应用实验报告

## 一、应用展示内容
本应用主题为 Art Space 数字艺术作品展示，基于 Jetpack Compose 实现作品浏览与交互切换功能。应用共展示 3 幅艺术作品，分别为《Life》《Flowers》《Sunflower》，每幅作品对应展示作者与创作年份信息。

## 二、界面结构说明
1. **界面区块划分**
- 标题展示区：展示应用名称
- 作品图片区：展示当前艺术作品，带画框效果
- 作品信息区：展示作品名称、作者、创作年份
- 按钮控制区：包含上一张、下一张切换按钮

2. **使用的可组合项**
使用 Column、Row 完成整体布局，使用 Image、Text 展示内容，使用 Button 实现交互，使用 Surface 实现画框样式，使用 Spacer 与内边距控制间距。

3. **组件嵌套结构**
以 Column 为根布局垂直排列所有内容，依次为标题文本、带画框的图片、作品信息区域、切换按钮区域，按钮区域使用 Row 水平排列。

## 三、Compose 状态管理实现
通过 Compose 状态管理机制记录当前显示的作品索引，程序会根据索引自动切换对应的图片、标题、作者与年份信息。状态发生变化时，界面会自动刷新更新内容，实现数据驱动界面展示。

## 四、切换按钮条件逻辑说明
1. **上一张按钮**
当前为第一幅作品时，点击后跳转到最后一幅作品；其他作品则向前切换一幅，实现循环浏览。

2. **下一张按钮**
当前为最后一幅作品时，点击后跳转到第一幅作品；其他作品则向后切换一幅，实现循环浏览。

## 五、遇到的问题与解决过程
1. **界面元素拥挤，布局不美观**
通过调整间距与内边距，优化各区域之间的距离，使界面布局更加整洁有序。

2. **图片展示效果单调**
使用画框样式修饰图片，增强艺术展示效果，使界面风格更贴合应用主题。

3. **按钮点击后界面不更新**
正确使用 Compose 状态管理方式，保证状态改变时界面可以自动重组刷新。

4. **文本信息层级不清晰**
通过设置不同的文字大小、粗细和颜色，区分标题与详情信息，提升界面可读性。

## 六、实验总结
本次实验完成了 Art Space 应用的开发，实现了艺术作品展示与循环切换功能。通过实验掌握了 Jetpack Compose 界面布局、组件使用、状态管理和交互逻辑编写，应用运行稳定、界面美观，满足全部实验要求。
