## 二、布局结构说明
1. 整体使用 `Column` 作为根布局，设置浅蓝背景色并全屏显示。
2. 界面分为上下两个区域：
   - 上方区域 `CardTop`：使用 `Column` 垂直排列头像、姓名、职位，整体水平居中。
   - 下方区域 `CardBottom`：使用 `Column` 放置三行联系方式，整体水平居中。
3. 每一行联系方式封装为 `ContactRow` 组件，使用 `Row` 横向排列图标与文字，实现图标+信息的样式。
4. 使用 `Spacer` 控制组件间距，使用 `Modifier` 实现尺寸、边距、对齐等样式控制。
   
## 三、遇到的问题及解决方法
1. 问题：应用启动后闪退，弹出一下就消失。
   解决：原因是图片资源名称不匹配或不存在，将图片正确放入 drawable 并修改对应资源 ID 后正常运行。

2. 问题：`Spacer` 使用 `padding` 导致界面异常。
   解决：将 `Modifier.padding` 改为 `Modifier.size`，间距正常显示。

3. 问题：联系方式靠左显示，不够美观。
   解决：在 `Column` 中添加 `horizontalAlignment = Alignment.CenterHorizontally`，实现整体居中。

4. 问题：联系方式位置偏下。
   解决：调整间距与布局排列方式，将联系方式整体上移，界面更加紧凑协调
