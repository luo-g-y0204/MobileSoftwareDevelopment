\# ArtSpace 艺术画廊应用开发报告



\## 一、项目概述

本项目基于 Android Jetpack Compose 框架，开发了一款简易艺术画廊应用（ArtSpace）。应用实现了图片轮播、作品信息展示、前后循环切换等核心功能，界面采用 Material Design 3 风格设计，整体布局简洁美观，交互逻辑清晰，可在 Android 模拟器或真机上正常运行。



\## 二、开发环境与技术栈

\- 开发工具：Android Studio

\- 开发语言：Kotlin

\- UI 框架：Jetpack Compose

\- 主题规范：Material Design 3

\- 适配模式：沉浸式全屏（EdgeToEdge）



\## 三、项目目标

1\.  实现艺术作品图片的卡片式展示，包含圆角、边框与阴影效果。

2\.  展示作品标题、作者、创作年份等基础信息。

3\.  支持「上一张/下一张」按钮控制，实现作品列表的循环切换。

4\.  界面布局居中对齐，适配主流手机屏幕尺寸，保证交互流畅。



\## 四、功能实现说明

\### 1. 作品展示模块

\- 采用卡片式布局展示艺术作品图片，添加圆角、内边距与阴影效果，增强视觉层次感。

\- 图片自适应缩放，保持比例填充卡片，避免拉伸变形。



\### 2. 作品信息模块

\- 在图片下方居中展示作品标题、作者与年份信息。

\- 标题采用加粗、大字号突出显示，作者与年份采用灰色小号字体，形成清晰的信息层级。



\### 3. 交互控制模块

\- 底部设置「Previous」与「Next」两个按钮，支持前后切换作品。

\- 实现循环切换逻辑：当前为第一张作品时，点击「Previous」将跳转到最后一张；当前为最后一张作品时，点击「Next」将回到第一张。



\### 4. 界面适配与优化

\- 采用沉浸式全屏模式，隐藏系统状态栏与导航栏，提升展示效果。

\- 界面元素均采用固定间距与居中对齐设计，适配不同分辨率的 Android 设备。



\## 五、项目特点

1\.  \*\*声明式 UI 开发\*\*：使用 Jetpack Compose 构建界面，无需 XML 布局文件，代码结构清晰，维护成本低。

2\.  \*\*状态驱动更新\*\*：通过 `remember` 管理当前作品索引状态，状态变化自动触发界面刷新，无需手动调用更新方法。

3\.  \*\*模块化组件设计\*\*：将图片展示、信息显示、控制按钮拆分为独立的可复用组件，便于后续扩展与维护。

4\.  \*\*循环交互逻辑\*\*：边界处理友好，避免了越界访问问题，提升用户体验。

5\.  \*\*规范的视觉设计\*\*：遵循 Material Design 3 设计规范，界面风格统一，视觉效果美观。



\## 六、运行效果

应用在 Android 模拟器（Medium Phone API 36.1）上运行正常，界面布局完整，按钮响应流畅，作品切换无卡顿，实现了全部预期功能。



\## 七、项目总结

本项目完成了 ArtSpace 艺术画廊应用的核心开发任务，实现了作品展示、信息呈现与交互控制的完整流程。代码结构清晰，符合 Jetpack Compose 开发规范，界面美观且交互友好，可直接作为课程设计或学习项目提交。
