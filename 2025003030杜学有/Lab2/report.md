## 二、布局结构简要说明
整个界面采用 Jetpack Compose 声明式 UI 编写，使用组件化思想将界面拆分为多个可组合函数，布局清晰、层次分明。

1. **BusinessCard（根布局）**
   - 使用 Column 作为根容器
   - 设置全屏大小、自定义浅绿色背景
   - 整体内容水平居中、垂直居中
   - 分为上半部分 CardTop 和下半部分 CardBottom

2. **CardTop（上半部分信息区）**
   - 使用 Column 垂直排列
   - 包含圆形头像、姓名、职位三个组件
   - 内容水平居中，文字样式区分明显

3. **CardBottom（下半部分联系方式区）**
   - 使用 Column 纵向排列
   - 包含三行联系方式，每行均使用 ContactRow 组件
   - 整体宽度填充屏幕，底部留有间距

4. **ContactRow（单行联系方式）**
   - 使用 Row 横向排列
   - 左侧为 Material 图标，右侧为文字信息
   - 图标与文字垂直居中对齐，样式统一规范

## 三、遇到的问题和解决过程
1. **圆形头像实现问题**
   - 问题：默认图片为矩形，不符合名片设计效果
   - 解决：使用 Modifier.clip(CircleShape) 将图片裁剪为圆形，配合 ContentScale.Crop 保证图片不变形。

2. **图标与文字对齐问题**
   - 问题：图标和文字高度不一致，显示不整齐
   - 解决：在 Row 中设置 verticalAlignment = Alignment.CenterVertically，实现自动垂直居中。

3. **界面样式统一问题**
   - 问题：图标与文字颜色杂乱，主题不统一
   - 解决：使用统一的绿色（0xFF3DDC84）设置图标颜色和职位文字颜色，使整体风格一致。

4. **布局间距与美观问题**
   - 问题：上下区域距离太近，界面拥挤
   - 解决：使用 Spacer 组件设置 100dp 高度，让上下部分间距合理，提升视觉效果。
